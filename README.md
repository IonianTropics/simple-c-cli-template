# example

Simple C and Makefile template

Designed to get you started with a C CLI tool with minimal overhead and room to grow

`example.c` contains some dead simple boilerplate code for a `cat` utility

## License

Your project can use whatever license you'd like. After setup head over to https://choosealicense.com.

## Setup

run `./setup.sh project-name "Your Name"` with your actual project name and Your Name

After setup, feel free to remove the script with `rm setup.sh`
